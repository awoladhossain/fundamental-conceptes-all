# ЁЯМР What is a Network?

`Network is a system where two or more devices are connected to communicate with each other.`

---

## ЁЯУж How is Data Transmitted?
Think of it like sending a letter to a friend:

1. You write the letter (data)
2. You put it in an envelope (packaging)
3. You write the address (IP address)
4. You give it to the post office (router)
5. It reaches your friend (destination)
The same logic applies in networks! Data is divided into small pieces called packets and sent to the destination.

## ЁЯФС Basic Concepts
Concept	Description
- Packet: 	Small piece of data
- IP Address: 	Device's address
- Router: 	Device that routes packets
- Protocol: 	Communication rules


ЁЯОп Summary
- Network = System of connected devices
- Data is transmitted in packets
- Each device has a unique address

## ЁЯУН What is IPv4?
The most commonly used IP format:

```bash

```
- 32-bit address (4 octets ├Ч 8 bits)
- Each octet takes values from 0-255
- Total of ~4.3 billion unique addresses
- Used since 1981
